## Projeto 👨‍💻
Este é um aplicativo de filmes para dispositivos móveis que permite aos usuários visualizar informações sobre filmes e assistir trailers.
## Tecnologias Usadas | Emulador 🏗️
 - Android Studio
 - React native 
    - Expo
    - vector-icons
    - youtube-iframe

- API The Movie DB

## Funcionalidades 🚀
- Busca de filme
- Listagem de filme
- Informações sobre filme
